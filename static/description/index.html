🌐 Network Quality Management

Gérez vos projets réseau & cybersécurité de A à Z

Module complet pour les intégrateurs réseau, consultants cybersécurité et ingénieurs IT. Couvre tout le cycle de vie d'un projet réseau : conception, déploiement, tests, audit et livraison.

Fonctionnalités principales

📁 Projets Réseau

Kanban, workflow par phases (Conception → Déploiement → Tests → Commissioning → Livraison), plan d'adressage IP intégré, gestion des équipements.

📋 Procédures Techniques

Bibliothèque de procédures réutilisables (VLAN, routage, hardening, firewall, VPN...), workflow de validation, commandes CLI intégrées.

✅ Checklists

Checklists par phase de projet avec suivi d'avancement, preuves attachables, items critiques bloquants et calcul automatique du taux de complétion.

📄 Documents de Configuration

21 types de documents (plans IP, configs Cisco, hardening, DOE client...), gestion de versions, workflow d'approbation, confidentialité.

🔒 Audits Sécurité

Constats classés par criticité (CVSS-like), score de sécurité automatique (/100), références CVE et MITRE ATT&CK, rapport exécutif + technique.

🚀 Commissioning

PV de mise en service formel avec tests de réception, résultats, réserves, signature électronique client + technicien.

Référencement automatique

Chaque objet possède une référence unique auto-générée :
NQM-0001 (Projets) · PROC-0001 (Procédures) · CHK-0001 (Checklists) · DOC-0001 (Documents) · AUD-0001 (Audits) · COM-0001 (Commissioning)
